# Crystal PingPong Example Shard

This is a simple, example shard.

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     pingpong:
       github: grokify/crystal-pingpong
   ```

2. Run `shards install`
